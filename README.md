## rayt

a raytracer written in Go , taken from [writing a raytracer in a weekend](https://raytracing.github.io/books/RayTracingInOneWeekend.html)

### Run Locally:

1. Install Go 1.22+
2. Install rayt as

```
git clone https://github.com/sagnikc395/rayt
```
3. Install ```just``` from homebrew:
```bash 
brew install just
```
4. Build the binary

```bash
just build 


```

5. Run and view the image
```bash 
just run
```
