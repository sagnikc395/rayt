## rayt 

writing a raytracer in go , taken from [writing a raytracer in a weekend](https://raytracing.github.io/books/RayTracingInOneWeekend.html)


### Run Locally:

1. Install Go 1.22+
2. Install rayt as 
```
go install github.com/sagnikc395/rayt
```
3. Build the binary
```bash 
make build 
```
4. Binary will be present in the ```./build/``` directory.
